_ds_bundle.js is the compiled, browser-ready version of all the design system components
 bundled into a single file.

 Source files (components/)	Bundle (_ds_bundle.js)
What you edit	What the browser runs
JSX + ES modules	Plain JS, no imports needed
Source of truth	Must be kept in sync manually
