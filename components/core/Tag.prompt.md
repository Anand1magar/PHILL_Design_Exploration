One-line: small label chip — six semantic colors, solid or outlined, optional leading/trailing icon and a stat count bubble.

```jsx
<Tag color="blue">Primary Insurance</Tag>
<Tag color="green" icon={<Icon name="check" size={10} />}>Verified</Tag>
<Tag color="red" variant="outlined">Rejected</Tag>
<Tag color="purple" stat={12}>Queue</Tag>
```

Colors: blue, gray, green, orange, purple, red. `variant`: solid (soft tint) / outlined. `size`: sm (10px) / lg (12px).
